Three Steps to Upload Your First Dompetitor

1) Download and Install a Python Interpreter

The first step in creating a Dompetitor is to install a Python interpretor -- a program that understands Python and can help you program and debug your code. Idle is a basic interpreter that comes bundled with a standard Python install. Download it here:

https://www.python.org/download/releases/2.7.8/
Once it finishes downloading, follow the necessary steps to install it.

(Obviously you can use any other interpreter if you have a preference)

2) Copy the Example Code

When you open Idle you should see two windows. One called the "Python Shell" and another for writing a new Python file which is probably labeled "Untitled" if you just installed. If you don't see the second window, open a new file with File>New or File>New Window.

The Python Shell is an interactive window where you can type python phrases and have them immediately interpretted. This is a great tool for touching up on your Python syntax or running a quick test. Go ahead and mess around with it for a few minutes.

Once you're done playing with the Python shell, open up your other editor window. Paste in the following code: show example

3) Upload Your File

Go the the upload page Click the "Choose File" button, find your file, and then click "Submit". Easy.

Python

Currently Dompetition is written in Python and if you want to participate you need to write code in Python -- but if you don't know Python, don't worry. Python could be the easiest language for those new to programming and its quick to pick up for those who already know how to program.

According to "PEP 20 (The Zen of Python)" the core philosophy of Python includes aphorisms such as:

  • Beautiful is better than ugly
  • Explicit is better than implicit
  • Simple is better than complex
  • Complex is better than complicated
  • Readability counts
